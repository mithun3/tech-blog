# Hello World

Every blog starts with a hello world post. This is mine.

## Why write?

Writing forces clarity. When I can't explain something simply, it means I don't understand it well enough yet. Publishing raises the bar — knowing someone else might read it makes me think twice before hand-waving past a detail I haven't fully worked out.

## What this is

This is a personal knowledge base and writing space. Posts will cover things I'm actively building or learning — web development, homelab infrastructure, tooling, and the occasional rabbit hole.

The [wiki section](/pages) is for reference material: notes and guides that get updated over time rather than stamped with a date and left alone.

## Stack

The blog itself is a good example of the kind of thing I'll be writing about:

- **Next.js 15** App Router
- **MDX** for content with React components in markdown
- **Tailwind CSS v4** for styling
- **Shiki** for syntax highlighting
- Deployed to **Vercel** (free tier, zero ops)

See [Building a Blog with Next.js 15 and MDX](/blog/building-with-nextjs) for the full breakdown.
