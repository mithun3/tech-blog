# Virtualization

If networking is the plumbing, virtualization is the floor plan. It is the part of a homelab that turns one box into several useful machines without making the whole thing feel improvised.

This section stays focused on compute, isolation, storage behavior, and day-2 operations. When a topic slides into bridges, DNS, or router behavior, it points back to the Networking section instead of repeating the same explanation with different nouns.
