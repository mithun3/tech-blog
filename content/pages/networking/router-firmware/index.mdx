# ASUS Merlin - Router Firmware

Merlin unlocks automation, scripting, and advanced routing capabilities that stock firmware does not expose. This section breaks the firmware features into setup, automation, packages, VPN policy routing, and DNS customization.

- [Merlin Setup](/pages/networking/router-firmware/merlin-setup) - stock vs Merlin, installation, updates, and rollback.
- [Router Scripting](/pages/networking/router-firmware/router-scripting) - event hooks, boot-time automation, cron jobs, and script troubleshooting.
- [Entware Packages](/pages/networking/router-firmware/entware-packages) - package installation, common commands, and useful add-ons.
- [VPN Director](/pages/networking/router-firmware/vpn-director) - selective routing by device and how to debug policy rules.
- [dnsmasq DNS Configuration](/pages/networking/router-firmware/dnsmasq-dns-config) - local records, network-wide blocking, and per-device DNS control.
